# EASYDAY
Елеусизов Абылай 
Федик Игорь
11-ВТИ-23
EasyDay предназначено для удобного планирования ежедневных задач и напоминаний. Приложение решает проблему забытых дел и плохой организации времени. Оно создано для школьников, студентов и занятых людей, которым нужен простой и понятный инструмент для эффективного управления своим днём.
