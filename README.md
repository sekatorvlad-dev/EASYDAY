# EASYDAY
Елеусизов Абылай 
Федик Игорь
11-ВТИ-23
EasyDay — это простое и удобное приложение для планирования повседневных задач и напоминаний. Оно помогает пользователям эффективно организовывать свой день, не забывать важные дела и встречи. Минималистичный интерфейс и push-уведомления делают приложение понятным, быстрым и комфортным в использовании.
